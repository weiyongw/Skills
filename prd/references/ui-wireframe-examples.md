# ASCII 线框图示例

> 用字符画出页面的大致布局，对齐"元素有哪些、放在哪、层次关系是什么"。
>
> 这不是设计稿，不需要像素级精确。目的是让产品、设计、研发对页面结构达成共识。

---

### 示例 1：看板风格的任务管理页

```text
+-----------------------------------------------------------------------------------+
| 项目仪表盘                                                       [用户头像] [v]   |
+-----------------------------------------------------------------------------------+
| [搜索...]  [按人员筛选 v]                                          [+ 新建任务]   |
+-----------------------------------------------------------------------------------+
|                                                                                   |
|  +-- 待处理 (3) --------+  +-- 进行中 (2) --------+  +-- 已完成 (5) --------+    |
|  |                       |  |                       |  |                       |   |
|  | US-101: 登录与验证    |  | US-203: 导航入口      |  | US-301: 权限设置      |   |
|  | #功能 [高]            |  | #杂务 [中]            |  | #基建 [低]            |   |
|  | 负责人: 爱丽丝        |  | 负责人: 鲍勃          |  | 负责人: 卡萝          |   |
|  |                       |  |                       |  |                       |   |
|  | ISSUE-102: 修复崩溃   |  |                       |  |                       |   |
|  | #缺陷 [紧急]          |  |                       |  |                       |   |
|  | 负责人: 鲍勃          |  |                       |  |                       |   |
|  |                       |  |                       |  |                       |   |
|  +-----------------------+  +-----------------------+  +-----------------------+   |
|                                                                                   |
+-----------------------------------------------------------------------------------+
```

### 示例 2：带筛选和分页的数据表格

```text
+-----------------------------------------------------------------------------------+
| 用户管理                                                              [导出 CSV]  |
+-----------------------------------------------------------------------------------+
| 状态: [激活 v]  角色: [所有角色 v]  日期: [起始] 至 [结束]              [查询]     |
+-----------------------------------------------------------------------------------+
|                                                                                   |
| +----+--------------+------------------------+--------+--------------+            |
| | ID | 姓名         | 邮箱                   | 角色   | 最后登录     |            |
| +----+--------------+------------------------+--------+--------------+            |
| | 12 | 爱丽丝       | alice@example.com      | 管理员 | 2025-09-20   |            |
| | 15 | 鲍勃         | bob@example.com        | 编辑   | 2025-09-19   |            |
| | 21 | 卡萝         | carol@example.com      | 查看者 | 2025-09-20   |            |
| +----+--------------+------------------------+--------+--------------+            |
|                                                                                   |
| 每页: [20 v]                           << 上一页 | 第 2 页 / 共 15 页 | 下一页 >> |
+-----------------------------------------------------------------------------------+
```

### 示例 3：表单页面（左右分栏）

```text
+-----------------------------------------------------------------------------------+
| 新建工单                                                                          |
+-----------------------------------------------------------------------------------+
|                                                                                   |
|  工单标题:  [____________________________________]                                |
|                                                                                   |
|  +-- 基本信息 ---------------+  +-- 详细描述 ---------------+                    |
|  |                            |  |                            |                   |
|  |  类型:    [Bug v]          |  |  [多行文本输入框]          |                   |
|  |  优先级:  [高 v]           |  |  [                        ]|                   |
|  |  负责人:  [选择... v]      |  |  [                        ]|                   |
|  |  截止日期: [YYYY-MM-DD]    |  |  [                        ]|                   |
|  |                            |  |                            |                   |
|  +----------------------------+  +----------------------------+                   |
|                                                                                   |
|  附件: [拖拽上传或点击选择]                                                        |
|  - bug_screenshot.png  [删除]                                                     |
|                                                                                   |
+-----------------------------------------------------------------------------------+
| [取消]                                                              [提交工单]    |
+-----------------------------------------------------------------------------------+
```
